1. 이름 EditText 
기본키보드 한글키보드로 설정.

2. 아이디 회원가입 입력시
데이터베이스 안에 중복 아이디가 있는지 중복 버튼으로 확인하고
중복이면 아이디 생성이 안되게. - 완료

3. 생년월일은 (직접입력, 캘린더)로 설정할 수 있도록.

4. 비밀번호 보임 안보임표시.
   1) 비밀번호 설정시 문자 + 숫자로 설정하게
   2) 최소 비밀번호 8자 이상
이에 해당이 안된다면 textview(비밀번호) 옆에 붉은 글씨로 error 표시

5. 비밀번호 확인
 1) 보임 안보임 표시
 2) 입력 후 확인을 누르고 비밀번호와 같지
     않다면 다시 입력해주세요 Toast message 표시
     그리고 (textview) 비밀번호 확인 옆에 붉은 글씨로 error 표시

6. 회원가입 버튼
1) 회원가입 버튼을 눌렀을 시 모든 EditText에 문제가 없을 시 
   회원가입을 완료했습니다 Toast.message 표시
2) 하나라도 문제가 있을 시 회원가입에 실패했습니다
   Toast.message 표시
